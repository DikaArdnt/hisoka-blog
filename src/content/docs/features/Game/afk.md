kamu bisa mengubah statusmu menjadi `AFK` agar seseorang tau kamu sedang tidak aktif dan sedang tidak ingin diganggu

|                       Info                        |      |
| :-----------------------------------------------: | :--: |
|                    Allow Basic                    |  ✅  |
|                Menggunakan Prefix                 |  ✅  |
|                 Menggunakan limit                 |  ✅  |
|                    Hanya Owner                    |  ❌  |
|                   Hanya Premium                   |  ❌  |
|                    Hanya Group                    |  ✅  |
|                 Hanya Admin Group                 |  ❌  |
|                Hanya Private Chat                 |  ❌  |
|                       Limit                       |  0   |
|                     Cooldown                      |  0   |
|                       Level                       |  0   |

# Command
## **#afk**

Pengguna dapat mengirimkan perintah bersamaan alasan, atau membalas media, misal
```sh
#afk mabar
```
