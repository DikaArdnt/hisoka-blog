# Command
## **#status**
Menampilkan daftar status dalam kurun waktu 24 jam

## **#getstatus**
Mendapatkan konten status dari target

## **#delstatus**
Menghapus status bedasarkan daftar

## **#upstatus**
Menambahkan status (Bisa sinkron berdasarkan privasi yang sudah diatur)
